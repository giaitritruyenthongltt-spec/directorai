# Social launch templates (P4.42)

Drop-in copy for every channel we post on launch day. Each file is
markdown so you can paste into Twitter, HN, Reddit, Discord without
manual reformatting.

| File                | Channel                    | Best post time (UTC)  |
| ------------------- | -------------------------- | --------------------- |
| `twitter-launch.md` | Twitter / X thread         | 14:00 UTC (09:00 EST) |
| `hn-launch.md`      | Hacker News (Show HN)      | 14:15 UTC             |
| `reddit-editors.md` | r/editors                  | 14:30 UTC             |
| `discord-launch.md` | Own Discord #announcements | 14:45 UTC             |

Use [twub](https://twub.app) or similar to spread the thread across
several tweets without losing the chain.

## Posting principles

- One link per platform. Don't cross-link from HN → Twitter; HN
  flags it.
- Reply within 15 minutes to every comment for the first 2 hours.
- No marketing language. Read the journalist FAQ tone if in doubt.
- Pin the launch thread to your profile for 48h.
