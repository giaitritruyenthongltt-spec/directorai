## Production notes

- **Capture**: 1080p60, OBS or Premiere screen capture. Mic = condenser
  with `audio.muteTrack` rule "no keyboard clack within 200ms of a cut".
- **Hosting**: YouTube unlisted first; embed in the docs site
  (P4.27) and the marketing site (P4.39). Promote to "public" on
  launch day.
- **Captions**: required (accessibility + SEO). Auto-generate via
  YouTube, hand-fix the technical terms.
- **Outro**: 3-second card with `directorai.app` + Discord link.
- **Pre-roll**: skip — keep the user inside the panel as fast as
  possible.
