# @directorai/shared

Cross-cutting utilities used by all DirectorAI packages.

## Exports

- `logger` — pino-based structured logger
- `errors` — typed `DirectorAIError` hierarchy
- `utils` — `sleep`, `retry`, `clamp`, `deepFreeze`, `uniqueId`

## Layer

Layer 1 (Infrastructure). Has no dependencies on other DirectorAI packages.
