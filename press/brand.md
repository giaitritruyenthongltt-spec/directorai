# Brand — quick reference

| Warning / P0          | `#c44`                               |                                   |

## Typography

We don't bundle a custom font — DirectorAI uses the system stack so
the panel matches Premiere's chrome and the marketing site adapts to
the reader's OS.

```
system-ui, -apple-system, "Segoe UI", Roboto, sans-serif
```

Headings use the same family with weight 600–700.

For monospace (code, license keys, file paths):

```
ui-monospace, "Cascadia Mono", "JetBrains Mono", Menlo, monospace
```

## Voice

- **Direct.** No "leverage AI to revolutionize your workflow".
- **Concrete.** Examples + numbers over adjectives.
- **Modest.** We deliver, then describe.
- **Honest about limits.** macOS isn't shipping yet; we say so.

## What we don't do

- No bro / hustle language.
- No "world's first" / "game-changing".
- No animated splash screens.
- No fake testimonials. Real beta quotes only, attributed when
  permitted.
