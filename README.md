# DirectorAI

> **AI Editing Copilot for Adobe Premiere Pro** — UXP plugin + MCP server + Python context engine, orchestrated by Claude.

## What is it?

DirectorAI connects Claude to Adobe Premiere Pro and gives it three superpowers:

1. **Control** — drive timeline edits, effects, color, audio, export via natural language.
2. **Understand** — analyze footage (transcribe, vision, scene, beat) to know what's in your clips.
3. **Style** — learn your editing style from past projects and apply it to new footage.
